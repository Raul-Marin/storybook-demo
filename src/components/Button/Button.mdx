# Button

Componente Button sincronizado con Figma.

## Código del Componente

```jsx

// Uso básico
<Button label="Button" />

// Con todas las props

  primary={true}
  size="medium"
  label="Button"
  onClick={() => console.log('Click!')}
/>
```
